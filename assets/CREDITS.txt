Asset receipts for the Flush Rush companion assets — the chapter 11
habit, practiced: what it is, what made it, what the license says.

MUSIC (assets/music/)
theme.ogg — gameplay loop. Generated locally with Meta MusicGen
  (musicgen-stereo-large) via tools/generate_music.py on the author's
  GPU. Weights license: CC-BY-NC 4.0 — NON-COMMERCIAL. Fine for the
  free companion game and your learning projects; NOT for a game you
  sell. Regenerate with a commercially-licensed model before any paid
  release (chapter 15 explains when this bites).
menu.ogg — menu loop. Same provenance and license as theme.ogg.

SOUND EFFECTS (assets/sfx/)
pop.wav, whoosh.wav, sting.wav, click.wav, zing.wav — synthesized
  (no AI) with tools/generate_sfx.py. Public domain. Go nuts.

SPRITES (assets/sprites/)
duck.png, plunger.png, bubble.png, logo-flush-rush.png, clogged.png,
icon-duck-face.png — generated with OpenAI's image API (gpt-image-1,
  transparent background) via tools/generate_sprite.py, directed and
  curated by the author. Per OpenAI's terms, the user owns generated
  outputs. Disclose AI generation where a store asks (chapter 15).

VOICE (assets/voice/)
title.ogg, clogged.ogg, highscore.ogg — deadpan announcer lines
  generated locally with Qwen3-TTS (12Hz 1.7B CustomVoice) on the
  author's GPU via a qwen_tts script; style-instructed synthesis
  ("deadpan, dry, unimpressed game announcer"). Check the model's
  license before commercial use; disclose as AI voice where asked.
  Not wired into the game — chapter 11 keeps the voice optional.

BACKGROUND (assets/sprites/background-pipe*.png)
  Pipe interior generated with gpt-image-2 via tools/generate_sprite.py
  (--opaque), muted per chapter 12's readability law. Same terms as
  the other image assets.
